This page provides links to tools that can be used to inspect and improve your experiences with webmaps generated with the ArcGIS JavaScript API. To use these tools:

  1. Save the bookmarklets to your browser bookmarks.
  2. Navigate to a map created with the ArcGIS JavaScript API
  3. Select the bookmarklet from the list.

Dependencies

  • Desktop browser (IE8+, Chrome, or Firefox. Untested on Safari for Mac).

Tools

  • Map Effects

    Drunk Map
    Adds an extra delay and slowness when moving around on the map. Works with both LeafletJS and ArcGIS JavaScript API maps.
